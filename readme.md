## 📖 О проекте

**VirtualArduino** — это программный комплекс, объединяющий в себе:
-  **Редактор кода** с подсветкой синтаксиса Arduino C++
-  **Эмулятор микроконтроллера** ATmega328P
-  **Визуальный редактор схем** с библиотекой компонентов
-  **Виртуальный монитор порта** для отладки

Приложение предназначено для образовательных целей и позволяет изучать основы программирования микроконтроллеров, принципы работы цифровых и аналоговых сигналов, а также проектировать и отлаживать электрические схемы без приобретения физического оборудования на начальном этапе обучения.

### Основные возможности:
- Создание, открытие и сохранение проектов в собственном формате (.vap)
- Подсветка синтаксиса и базовая проверка ошибок в коде
- Эмуляция работы с цифровыми и аналоговыми пинами (INPUT, OUTPUT, INPUT_PULLUP)
- Поддержка функций `delay()`, `millis()`, `Serial.print()`
- Визуализация работы светодиодов, резисторов, моторов и других компонентов
- Масштабируемое рабочее поле с привязкой к сетке
- Реакция схемы на выполняемую программу в реальном времени

## 📥 Установка и запуск

### Требования
- **Python 3.10** или выше
- **Git** (для клонирования репозитория)
- **Windows** или **Linux**

### Инструкция для PowerShell (Windows)

```powershell
# 1. Клонируйте репозиторий
git clone https://github.com/deepS7S/Virtual-Arduino.git
cd VirtualArduino

# 2. Создайте виртуальное окружение
python -m venv .venv

# 3. Активируйте виртуальное окружение
.venv\Scripts\Activate.ps1

# 4. Установите зависимости из requirements.txt
pip install -r requirements.txt

# 5. Запустите приложение
python main.py
```

### Инструкция для Linux (Bash)

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/deepS7S/Virtual-Arduino.git
cd VirtualArduino

# 2. Создайте виртуальное окружение
python3 -m venv .venv

# 3. Активируйте виртуальное окружение
source .venv/bin/activate

# 4. Установите зависимости
pip install -r requirements.txt

# 5. Запустите приложение
python3 main.py
```
